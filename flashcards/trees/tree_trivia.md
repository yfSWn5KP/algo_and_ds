### How many nodes are in a tree?
num_nodes == ?

num_nodes == branches<sup>height</sup> - 1
>
> branches == number of possible forking options at each node
> 
> height == total height of the tree
>
This binary tree has 2<sup>3</sup> - 1 == 7 nodes
```
     O
   /   \
  O     O
 / \   / \
O   O O   O
```
